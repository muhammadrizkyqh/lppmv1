# === SETUP DNS DI CLOUDFLARE untuk stai-ali.com ===

## 1. Login ke Cloudflare Dashboard
https://dash.cloudflare.com/

## 2. Pilih domain "stai-ali.com"

## 3. Klik menu "DNS" di sidebar kiri

## 4. Tambahkan A Record:

### Record 1: Root domain
- Type: A
- Name: @ (atau stai-ali.com)
- IPv4 address: 195.88.211.123
- Proxy status: DNS only (ikon abu-abu) ← PENTING untuk SSL Certbot
- TTL: Auto

### Record 2: WWW subdomain
- Type: A
- Name: www
- IPv4 address: 195.88.211.123
- Proxy status: DNS only (ikon abu-abu) ← PENTING untuk SSL Certbot
- TTL: Auto

## 5. Save/Add Record

## 6. PENTING: Matikan Cloudflare Proxy dulu
Klik ikon oranye (proxied) jadi abu-abu (DNS only) untuk kedua record.
Ini perlu agar Certbot bisa verify domain untuk SSL.

## 7. Tunggu DNS propagation (5-10 menit)

## 8. Test DNS sudah resolve:
```
ping stai-ali.com
ping www.stai-ali.com
```
Harusnya resolve ke 195.88.211.123

## 9. Setelah SSL berhasil diinstall (langkah di VPS selesai),
    Anda BISA aktifkan Cloudflare Proxy lagi (ikon oranye)
    untuk mendapat:
    - DDoS protection
    - CDN caching
    - WAF (Web Application Firewall)

## CATATAN:
- Jangan aktifkan proxy SEBELUM SSL terinstall
- Setelah SSL aktif, proxy bisa dinyalakan untuk extra security
- Cloudflare akan handle SSL certificate mereka sendiri
